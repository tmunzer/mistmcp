Recommended

Use Hosted MCP Server

Connect directly to our hosted MCP server at mcp.mist-lab.fr - no installation required!

1

Get Your Mist API Token

Visit Mist Dashboard ‚Üí Organization ‚Üí Settings ‚Üí API Tokens to create a new token.

2

Configure Claude Desktop

Add to your claude_desktop_config.json:

{
  "mcpServers": {
    "mist-http": {
      "command": "npx",
        "args": [
          "-y", 
          "mcp-remote",
          "https://mcp.mist-lab.fr/mcp/?cloud=api.mist.com",
          "--header",
          "X-Authorization:${MIST_APITOKEN}",
          "--transport",
          "http-only"
        ],
        "env": {
          "MIST_APITOKEN": "your-api-token"
        }
      }
  }
}
3

Start Using

Restart Claude Desktop and start managing your Mist network with AI!

Local Installation

Install and run the MCP server locally for full control and customization.

1

Clone & Install

Download and install the MCP server from GitHub

# Clone the repository
git clone https://github.com/tmunzer/mistmcp.git
cd mistmcp

# Install dependencies (using uv - recommended)
uv sync

# Or using pip
pip install -e .
2

Configure Environment

Set up your Mist API credentials

# Create .env file
MIST_API_TOKEN=your_mist_api_token_here
MIST_HOST=api.mist.com  # or your custom host
3

Configure Claude Desktop

Add to your claude_desktop_config.json:

{
  "mcpServers": {
    "mist-local": {
      "command": "uv",
      "args": [
        "--directory",
        "/absolute/path/to/mistmcp",
        "run",
        "mistmcp"
      ],
      "env": {
        "MIST_ENV_FILE": ".env"
      }
    }
  }
}
4

Test & Start Using

uv run mistmcp --transport http --port 8000
